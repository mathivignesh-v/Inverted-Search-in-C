# Inverted Search Using Data Structures

## Overview

Inverted Search is a C-based project developed using Data Structures to perform fast and efficient word searching across multiple files. The project creates an inverted index that maps words to the files in which they appear, improving search efficiency and reducing search time.

## Features

* Create and update inverted index
* Search words across multiple files
* Display indexed data
* File handling support
* Efficient searching using linked lists and hashing
* Duplicate word handling

1. Read data from multiple text files
